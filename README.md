# Mind Mart

A Java-powered game that raises mental health awareness. Users are prompted to interact with different mental health scenarios, purchase products, and make diagnostic inferences.

## How to run
1. Clone this repo
2. Have Java Runtime Environment (JRE) or Java Development Kit (JDK) installed
    Install:
     `sudo apt install default-jdk # for JDK`
      `sudo apt install default-jre # for JRE`
  
     Check:
     `java -version`
     `javac -version`
3. Click the Run button (▶️) in VS Code
